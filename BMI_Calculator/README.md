# ⚖️ BMI Calculator (Python GUI)

## 🚀 Overview

This project is a **BMI (Body Mass Index) Calculator** built using Python with a graphical user interface (GUI).
It allows users to input their weight and height (in cm), calculates BMI, and displays the corresponding health category.

This project was developed as part of the **Oasis Infobyte Python Internship**.

## ✨ Features

* 📥 User Input (Weight & Height in cm)
* ⚖️ BMI Calculation
* 📊 BMI Category Classification
* 🖥️ GUI Interface using Tkinter
* 💾 Data Storage using CSV
* 📜 View BMI History
* 📈 Graph Visualization (BMI Trends)
* ⚠️ Error Handling


## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* CSV (Data Storage)
* Matplotlib (Graph Visualization)

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```
git clone https://github.com/NIKHILGAJA/-OIBSIP
cd bmi-calculator
```

### 2️⃣ Install Requirements

```
pip install -r requirements.txt
```

### 3️⃣ Run Application

```
python bmi_calculator.py
```

## 🎯 Usage

1. Enter weight (kg)
2. Enter height (cm)
3. Click "Calculate BMI"
4. View result and category
5. Use:

   * "View History" to see saved data
   * "Show Graph" to visualize BMI trends
