# 🎙️ Voice Assistant with GUI (Python)

## 🚀 Overview

This project is a **Voice Assistant built using Python** with a simple graphical user interface (GUI).
It can listen to voice commands, process them, and respond with voice output while also displaying interactions on screen.

This project was developed as part of the **Oasis Infobyte Python Internship**.

---

## ✨ Features

* 🎤 Speech Recognition (Voice Input)
* 🔊 Text-to-Speech Response
* 🕒 Tell Time & Date
* 🌐 Open Websites (Google, YouTube, WhatsApp, etc.)
* 🔍 Google Search via Voice
* 📚 Wikipedia Search
* 🌦️ Weather Information (API-based)
* 🖥️ GUI Interface using Tkinter
* ⚠️ Error Handling for better user experience

---

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Tkinter (GUI)
* Wikipedia API
* OpenWeather API

## ▶️ How to Run the Project

Follow these steps to run the Voice Assistant on your system:

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/voice-assistant.git
cd voice-assistant
```

### 2️⃣ Create Virtual Environment (Recommended)

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Required Libraries

```
pip install -r requirements.txt
```

### 4️⃣ Install PyAudio (If Error Occurs)

```
pip install pipwin
pipwin install pyaudio
```

### 5️⃣ Add Weather API Key (Optional)

* Go to https://openweathermap.org/
* Generate API key
* Replace in code:

```
api_key = "YOUR_API_KEY"
```

### 6️⃣ Run the Application

```
python main.py
```
